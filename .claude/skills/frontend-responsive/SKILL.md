## When to use this skill

- When writing mobile-first CSS (base styles for mobile, then add `sm:`, `md:`, `lg:`)
- When implementing responsive grid layouts (`grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`)
- When sizing touch targets for mobile (minimum 44x44px)
- When using fluid containers (`container mx-auto px-4`)
- When adjusting typography for different screen sizes
- When building mobile navigation or bottom navigation bars
- When testing layouts across device sizes
- When optimizing images and content for mobile
