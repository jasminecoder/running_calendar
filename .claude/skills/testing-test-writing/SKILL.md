## When to use this skill

- When writing model tests (`test/models/*_test.rb`)
- When writing controller tests (`test/controllers/*_test.rb`)
- When writing system tests with Capybara (`test/system/*_test.rb`)
- When creating test fixtures (`test/fixtures/*.yml`)
- When using assertions (`assert`, `assert_equal`, `assert_includes`, `assert_not`)
- When running tests (`bin/rails test`, `bin/rails test:system`)
- When mocking external services in tests
- When testing validations and scopes
- When following test naming conventions
