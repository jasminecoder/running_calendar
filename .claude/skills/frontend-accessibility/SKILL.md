## When to use this skill

- When creating or editing ERB view templates (`app/views/**/*.html.erb`)
- When adding form inputs and labels
- When implementing keyboard navigation and focus states
- When writing image alt text
- When using ARIA attributes for complex components
- When structuring headings (h1-h6) properly
- When adding Tailwind focus ring classes
- When building accessible cards, buttons, or navigation
- When testing with screen readers
