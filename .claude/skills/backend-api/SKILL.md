## When to use this skill

- When creating or editing Rails controller files (`app/controllers/**/*.rb`)
- When defining or modifying routes in `config/routes.rb`
- When implementing RESTful CRUD actions (index, show, new, create, edit, update, destroy)
- When handling form submissions and redirects
- When setting up Turbo Stream responses for dynamic updates
- When using strong parameters to whitelist permitted attributes
- When adding before_action callbacks for authentication or resource loading
- When responding to multiple formats (HTML, Turbo Stream, JSON)
