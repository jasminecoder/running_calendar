- When adding Tailwind CSS classes to ERB templates

- When creating custom CSS with `@apply` in `application.css`
- When working with Tailwind's spacing, color, and typography scales
- When building grids and flexbox layouts
- When styling buttons, cards, forms, and navigation
- When editing `tailwind.config.js` for customizations
