## When to use this skill

- When creating or editing ERB partials (`app/views/**/_*.html.erb`)
- When extracting shared markup into reusable partials
- When passing locals to partials instead of instance variables
- When rendering collections with `render partial: ..., collection:`
- When writing Stimulus controllers (`app/javascript/controllers/*.js`)
- When using `data-controller`, `data-action`, `data-target` attributes
- When wrapping content in Turbo Frames for dynamic updates
- When building interactive UI components (toggles, modals, dropdowns)
