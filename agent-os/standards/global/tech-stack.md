## Tech Stack Reference

### Framework & Runtime
- **Framework**: Ruby on Rails 8.x
- **Language**: Ruby 3.3+
- **Package Manager**: Bundler (Ruby), npm (JavaScript/CSS)

### Frontend
- **JavaScript**: Hotwire (Turbo + Stimulus)
- **CSS**: Tailwind CSS
- **Server Rendering**: ERB templates

### Database & Storage
- **Database**: PostgreSQL
- **ORM**: Active Record
- **Caching**: Solid Cache (Rails 8)
- **File Storage**: Active Storage

### Testing
- **Framework**: Minitest
- **System Tests**: Capybara + Selenium

### Deployment
- **Hosting**: Hetzner VPS
- **Deployment**: Kamal
- **Background Jobs**: Solid Queue (Rails 8)

### Third-Party Services
- **Email**: SendGrid (future)
